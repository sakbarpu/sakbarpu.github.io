A Table Summarizing All Traditional Machine Learning Methods

Wouldn't it be cool if you could have ALL RELEVANT INFORMATION about TRADITIONAL MACHINE LEARNING models in one page? I have created a table which compiles high level details of some traditional machine learning models. Specifically the table discusses different classification and clustering models. For each model I discuss many feartures like the model space, search function and score function. The purpose is to provide a cheatsheat for experienced machine learning folks and NOT to teach these methods. The table is created using the material I learnt during the CS-573 Data Mining course at Purdue. Please shoot me an email if you find errors in the slide or need some clarification.
