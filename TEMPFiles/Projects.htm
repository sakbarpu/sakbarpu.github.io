3D MODELING OF SKELETAL STRUCTURE OF TREES

 

This project focuses on the robust 3-dimensional modeling of skeletal structure of trees. The branches of trees are considered as the cylinders and RANSAC technique is used to fit the cylindrical model for branches.

 

 

 

 

 

 

 

 

 

 

 

 

 

OBJECT DETECTION AND RECOGNTION USING BOOSTING AND DEEP NEURAL NETWORKS

 

This project is based on the implementation of two step detection and recognition system. For detection Boosting technique is used while for recognition the popular Deep Neural Networks are used. DNNs are very famous for vision processing tasks. The database used for evaluating the system is the LabelMe object images database.
